Detailed Description

Axis-aligned bounding box defined by its minimal and maximal vertices' positions. 

     _______________________
    /|                    /|
   / |                   / |
  |---------------------| < maxPos
  |  |                  |  |
  |  |                  |  |
  |  |                  |  |
  |  |                  |  |
  | /-------------------|-/
  |/ ^ minPos           |/
  -----------------------

The min position designates the point in [ -X; -Y; -Z ], and the max the point in [ +X; +Y; +Z ].
