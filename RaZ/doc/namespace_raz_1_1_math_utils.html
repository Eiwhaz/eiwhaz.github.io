Computes the cubic Hermite interpolation between two thresholds.

Any value below minThresh will return 0, and any above maxThresh will return 1. Between both thresholds, a smooth interpolation is performed. 

  1.0    |               |___
         |           .-~"|
         |         ,^    |
         |        /      |
         |       /       |
         |      /        |
         |    ,v         |
  0.0 ___|,.-"           |
         ^               ^
     minThresh       maxThresh

This is equivalent to GLSL's smoothstep function.

Template Parameters
TType to compute the interpolation with.
Parameters
minThreshMinimum threshold value.
maxThreshMaximum threshold value.
valueValue to be interpolated.
Returns
0 if value is lower than minThresh.
1 if value is greater than maxThresh.
The interpolated value (between 0 & 1) otherwise.
