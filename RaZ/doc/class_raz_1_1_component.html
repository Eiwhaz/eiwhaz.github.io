◆ getId()

template<typename Comp >
std::size_t Raz::Component::getId ( )
static

Gets the ID of the given component. It uses CRTP to assign a different ID to each component it is called with. This function will be compiled every time it is called with a different component type, incrementing the assigned index. Note that it must be called directly from Component, and a derived class must be given (Component::getId<DerivedComponent>()).
