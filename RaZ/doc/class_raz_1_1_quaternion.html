Detailed Description

template<typename T = float>
class Raz::Quaternion< T >

Quaternion representing a rotation in 3D space. Quaternions are used to avoid gimbal locks, present with Euler angles.

Template Parameters
TType of the values to be held by the quaternion.

Computes the norm of the quaternion. Calculating the actual norm requires a square root operation to be involved, which is expensive. As such, this function should be used if actual length is needed; otherwise, prefer computeSquaredNorm().

◆ computeSquaredNorm()

template<typename T = float>
constexpr T Raz::Quaternion< T >::computeSquaredNorm ( ) const
inlinenoexcept

Computes the squared norm of the quaternion. The squared norm is equal to the addition of all components (real & complexes alike) squared. This calculation does not involve a square root; it is then to be preferred over computeNorm() for faster operations.

Returns
Quaternion's squared norm.
