Calls a function in parallel over a collection. The collection is automatically split, giving a separate start/past-the-end iterator range to each thread.

Note
The container must either be a constant-size C array, or have public begin() & end() functions.
If using Emscripten this call will be synchronous, threads being unsupported with it for now.
