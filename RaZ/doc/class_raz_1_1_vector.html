◆ computeLength()

template<typename T, std::size_t Size>
constexpr float Raz::Vector< T, Size >::computeLength ( ) const
inline

Computes the length of the vector. Calculating the actual length requires a square root operation to be involved, which is expensive. As such, this function should be used if actual length is needed; otherwise, prefer computeSquaredLength().

Returns
Vector's length.

◆ computeSquaredLength()

template<typename T, std::size_t Size>
constexpr float Raz::Vector< T, Size >::computeSquaredLength ( ) const
inlinenoexcept

Computes the squared length of the vector. The squared length is equal to the dot product of the vector with itself. This calculation does not involve a square root; it is then to be preferred over computeLength() for faster operations.

Returns
Vector's squared length.
