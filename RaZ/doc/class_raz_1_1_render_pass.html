bool Raz::RenderPass::isValid ( ) const

Checks that every write buffer has an associated read buffer in each of its children.

Note
This doesn't check for precedency; if no subsequent pass exist, it is considered valid.
Returns
True if no child pass or if the render pass is valid, false otherwise.
