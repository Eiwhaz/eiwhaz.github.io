◆ getId()

template<typename SysT >
std::size_t Raz::System::getId
static

Gets the ID of the given system type. It uses CRTP to assign a different ID to each system type it is called with. This function will be instantiated every time it is called with a different type, incrementing the assigned index. Note that it must be called directly from System, and a derived class must be given (System::getId<DerivedSystem>()).

Template Parameters
