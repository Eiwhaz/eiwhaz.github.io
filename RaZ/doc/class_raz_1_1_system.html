◆ getId()

template<typename Sys >
std::size_t Raz::System::getId ( )
static

Gets the ID of the given system. It uses CRTP to assign a different ID to each system it is called with. This function will be compiled every time it is called with a different system type, incrementing the assigned index. Note that it must be called directly from System, and a derived class must be given (System::getId<DerivedSystem>()).

Template Parameters
